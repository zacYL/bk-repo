# 删除仓库
功能描述：删除仓库

### 请求地址
```
/repo-repository/api/repo/delete/{projectId}/{repoName}
```

### 请求方法
`DELETE`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| projectId     | string   | true       | 所属项目 |
| repoName     | string   | true       | 仓库名称 |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| forced     | boolean   | true       | 是否强制删除 |


#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
