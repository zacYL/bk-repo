# 删除文件或空目录
功能描述：删除文件或空目录

### 请求地址
```
/repo-generic/{projectId}/{repoName}/{fullPath}
```

### 请求方法
`DELETE`
### 请求参数

#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| projectId     | string   | true    | 项目ID |
| repoName     | string   | true   | 仓库名 |
| fullPath | string   | true   | 文件或目录的全路径 |

#### 查询字符串参数

| 字段 | 类型 | 必填 | 描述 |
| ---- | ---- | ---- | ---- |

#### 请求体

无

### 返回结果

| 字段    | 说明       |
| ------- | ---------- |
| code    | 返回码     |
| message | 错误信息   |
| data    | 数据       |
| traceId | 链路追踪id |

#### 响应体示例

```json
{
  "code":0,
  "message":null,
  "data":null,
  "traceId":""
}
```
