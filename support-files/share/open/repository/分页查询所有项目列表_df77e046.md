# 分页查询所有项目列表
功能描述：分页查询所有项目列表

### 请求地址
```
/api/project/search
```

### 请求方法
`GET`
### 请求参数

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| namePrefix     | string   | false       | 项目名前缀 |
| pageNumber     | integer   | false       | 当前页 |
| pageSize     | integer   | false       | 分页大小 |
