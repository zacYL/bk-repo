# 修改仓库配额
功能描述：修改仓库配额

### 请求地址
```
/api/repo/quota/{projectId}/{repoName}
```

### 请求方法
`POST`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| projectId     | string   | true       | 所属项目 |
| repoName     | string   | true       | 仓库名称 |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| quota     | integer   | true       | 仓库配额 |


#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |

### 返回结果
