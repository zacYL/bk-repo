# 软件源--批量 添加/删除 管理员
功能描述：软件源--批量 添加/删除 管理员

### 请求地址
```
/api/user/admin/batch/{admin}
```

### 请求方法
`PUT`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| admin     | boolean   | true       | 执行操作，true:代表添加为管理员；false: 删除管理员 |



#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| null     | array<null>   |  |
