# 删除用户/部门
功能描述：删除用户/部门

### 请求地址
```
/api/bksoftware/unit/{repoName}
```

### 请求方法
`DELETE`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| repoName     | string   | true       | 仓库名 |



#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| department     | array<string>   |  |
| user     | array<string>   |  |
