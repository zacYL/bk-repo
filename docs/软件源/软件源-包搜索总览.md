### 包搜索总览(repository)

* API: GET /api/package/search/overview?repoType={RepositoryType}&projectId={String}&packageName={String}

* 请求体

  此接口请求体为空

* 请求字段说明

  | 字段        | 类型   | 默认值 | 是否必传 | 说明     | Description  |
    | ----------- | ------ | ------ | -------- | -------- | ------------ |
  | repoType    | String | 无     | 是       | 仓库类型 | ResourceType |
  | packageName | String | 无     | 否       | 包名     | packageName  |


* 响应体

  ```json
  {
      "code": 0,
      "message": null,
      "data": {
        "list": [
          {
            "repoName": "maven-1",
            "packages": 3
          },
          {
            "repoName": "maven-2",
            "packages": 4
          }
        ],
        "sum": 7
      },
      "traceId": ""
  }
  ```
