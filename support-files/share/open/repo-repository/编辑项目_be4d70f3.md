# 编辑项目
功能描述：编辑项目

### 请求地址
```
/repo-repository/api/project/{name}
```

### 请求方法
`PUT`
### 请求参数
#### 路径参数

| 字段   | 类型     | 必填  | 描述  |
|------|--------|-----|-----|
| name | String | 是   | 项目名 |

#### 请求体参数
| 字段          | 类型     | 必填  | 描述  |
|-------------|--------|-----|-----|
| description | String | 否   | 描述  |
| displayName | String | 否   | 显示名 |

### 返回结果

| 字段      | 说明     |
|---------|--------|
| code    | 返回码    |
| message | 错误信息   |
| data    | 数据     |
| traceId | 链路追踪id |

#### 响应体示例

```json
{
  "code": 0,
  "message": null,
  "data": true,
  "traceId": null
}
```

#### data 字段说明

是否修改成功
