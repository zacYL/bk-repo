# 查询项目是否存在
功能描述：查询项目是否存在

### 请求地址
```
/repo-repository/api/project/exist/{projectId}
```

### 请求方法
`GET`
### 请求参数
#### 路径参数

| 字段        | 类型     | 必填  | 描述     |
|-----------|--------|-----|--------|
| projectId | String | 是   | 项目名    |

### 返回结果

| 字段      | 说明     |
|---------|--------|
| code    | 返回码    |
| message | 错误信息   |
| data    | 数据     |
| traceId | 链路追踪id |

#### 响应体示例

```json
{
  "code": 0,
  "message": null,
  "data": true,
  "traceId": null
}
```

#### data 字段说明

项目是否存在
