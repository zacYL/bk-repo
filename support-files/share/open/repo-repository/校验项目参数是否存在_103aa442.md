# 校验项目参数是否存在
功能描述：校验项目参数是否存在

### 请求地址
```
/repo-repository/api/project/exist
```

### 请求方法
`GET`
### 请求参数

#### 查询参数

| 字段          | 类型     | 必填  | 描述  |
|-------------|--------|-----|-----|
| name        | String | 否   | 项目名 |
| displayName | String | 否   | 显示名 |

#### 响应体示例

```json
{
  "code": 0,
  "message": null,
  "data": true,
  "traceId": null
}
```

#### data 字段说明

项目参数是否存在
