# 查询制品状态
功能描述：查询制品状态

### 请求地址
```
/repo-repository/api/stage/{projectId}/{repoName}
```

### 请求方法
`GET`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| projectId     | string   | true       | projectId |
| repoName     | string   | true       | repoName |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| packageKey     | string   | true       | packageKey |
| version     | string   | true       | version |
