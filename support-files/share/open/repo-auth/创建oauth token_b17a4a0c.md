# 创建oauth token
功能描述：创建oauth token

### 请求地址
```
/api/oauth/token
```

### 请求方法
`POST`
### 请求参数

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| clientId     | string   | false       | clientId |
| clientSecret     | string   | false       | clientSecret |
| code     | string   | false       | code |
