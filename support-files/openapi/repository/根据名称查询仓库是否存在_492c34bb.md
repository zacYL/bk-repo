# 根据名称查询仓库是否存在
功能描述：根据名称查询仓库是否存在

### 请求地址
```
/api/repo/exist/{projectId}/{repoName}
```

### 请求方法
`GET`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| projectId     | string   | true       | 所属项目 |
| repoName     | string   | true       | 仓库名称 |
