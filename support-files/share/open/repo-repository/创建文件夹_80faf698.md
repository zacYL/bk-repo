# 创建文件夹

功能描述：创建文件夹

### 请求地址

```
/repo-repository/api/node/mkdir/{projectId}/{repoName}/**
```

### 请求方法

`POST`

### 请求参数

#### 路径参数

| 字段        | 类型     | 必填  | 描述            |
|-----------|--------|-----|---------------|
| projectId | String | 是   | 项目id          |
| repoName  | String | 是   | 仓库名称          |
| fullPath  | String | 否   | 完整路径, 为空则是根路径 |

### 返回结果

| 字段      | 说明     |
|---------|--------|
| code    | 返回码    |
| message | 错误信息   |
| data    | 数据     |
| traceId | 链路追踪id |

#### 响应体示例

```json
{
  "code": 0,
  "message": null,
  "data": null,
  "traceId": null
}
  ```
