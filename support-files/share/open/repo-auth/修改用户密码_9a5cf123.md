# 修改用户密码
功能描述：修改用户密码

### 请求地址
```
/api/user/update/password/{uid}
```

### 请求方法
`PUT`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| uid     | string   | true       | uid |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| oldPwd     | string   | true       | oldPwd |
| newPwd     | string   | true       | newPwd |


#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
