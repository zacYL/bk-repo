# 批量新增用户所属角色
功能描述：批量新增用户所属角色

### 请求地址
```
/api/user/role/add/{rid}
```

### 请求方法
`PATCH`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| rid     | string   | true       | 用户角色Id |



#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| null     | array<null>   |  |
