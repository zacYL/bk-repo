# 查询项目下有权限的部门列表: CI 项目下有权限的部门
功能描述：查询项目下有权限的部门列表: CI 项目下有权限的部门

### 请求地址
```
/api/department/list/{projectId}
```

### 请求方法
`GET`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| projectId     | string   | true       | 项目ID |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| userId     | string   | false       | userId |
