# 删除项目
功能描述：删除项目

### 请求地址
```
/repo-repository/api/project/delete/{name}
```

### 请求方法
`DELETE`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| name     | string   | true       | 项目ID |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| confirmName     | string   | true       | 用户输入的确认信息(项目id) |


#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
