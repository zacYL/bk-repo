# 批量创建用户
功能描述：批量创建用户

### 请求地址
```
/api/user/batch
```

### 请求方法
`POST`
### 请求参数



#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |

### 返回结果
| 字段 | 类型 | 描述 |
| -------- | -------- | -------- |
| code     | integer , format:int32  | 返回码 |
| data     | BatchCreateUserResponse   | 数据 |
| message     | string   | 错误信息 |
| traceId     | string   | 链路追踪id |
#### BatchCreateUserResponse
| 字段 | 类型 | 描述 |
| -------- | -------- | -------- |
| failed     | integer , format:int32  |  |
| failedUsers     | array<string>   |  |
| success     | integer , format:int32  |  |
