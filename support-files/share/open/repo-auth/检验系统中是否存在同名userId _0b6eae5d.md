# 检验系统中是否存在同名userId 
功能描述：检验系统中是否存在同名userId 

### 请求地址
```
/api/user/repeat/{uid}
```

### 请求方法
`GET`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| uid     | string   | true       | uid |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| userId     | string   | false       | userId |
