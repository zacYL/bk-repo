# 查询公钥列表
功能描述：查询公钥列表

### 请求地址
```
/api/key/list
```

### 请求方法
`GET`




### 返回结果
| 字段 | 类型 | 描述 |
| -------- | -------- | -------- |
| code     | integer , format:int32  | 返回码 |
| data     | array<密钥>   | 数据 |
| message     | string   | 错误信息 |
| traceId     | string   | 链路追踪id |
#### 密钥
| 字段 | 类型 | 描述 |
| -------- | -------- | -------- |
| createAt     | string , format:date-time  | 创建日期 |
| fingerprint     | string   | 指纹 |
| id     | string   | id |
| name     | string   | 名称 |
