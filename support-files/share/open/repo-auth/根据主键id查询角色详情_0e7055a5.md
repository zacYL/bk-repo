# 根据主键id查询角色详情
功能描述：根据主键id查询角色详情

### 请求地址
```
/api/role/detail/{id}
```

### 请求方法
`GET`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| id     | string   | true       | 角色主键id |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| userId     | string   | false       | userId |



### 返回结果
| 字段 | 类型 | 描述 |
| -------- | -------- | -------- |
| code     | integer , format:int32  | 返回码 |
| data     | 角色   | 数据 |
| message     | string   | 错误信息 |
| traceId     | string   | 链路追踪id |
#### 角色
| 字段 | 类型 | 描述 |
| -------- | -------- | -------- |
| admin     | boolean   | 管理员 |
| description     | string   | 描述信息 |
| id     | string   |  |
| name     | string   | 角色名 |
| projectId     | string   | 项目ID |
| repoName     | string   | 仓库名称 |
| roleId     | string   | 角色ID |
| type     | string   | 角色类型 |
| users     | array<string>   | 绑定的用户 |
