# 用户分页列表
功能描述：用户分页列表

### 请求地址
```
/api/user/page/{pageNumber}/{pageSize}
```

### 请求方法
`GET`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| pageNumber     | integer   | true       | pageNumber |
| pageSize     | integer   | true       | pageSize |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| userId     | string   | false       | userId |
| user     | string   | true       | user |
| admin     | boolean   | true       | admin |
| locked     | boolean   | true       | locked |
