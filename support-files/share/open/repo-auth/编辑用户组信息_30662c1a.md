# 编辑用户组信息
功能描述：编辑用户组信息

### 请求地址
```
/api/role/{id}
```

### 请求方法
`PUT`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| id     | string   | true       | id |



#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| description     | string   |  |
| name     | string   |  |
| userIds     | array<string>   |  |
