# 审计日志事件类型
功能描述：审计日志事件类型

### 请求地址
```
/repo-repository/api/log/event/type
```

### 请求方法
`GET`

### 请求参数

#### 请求头参数

| 字段                  | 类型   | 必填 | 描述                  |
| --------------------- | ------ | ---- | --------------------- |
| X-DEVOPS-ACCESS-TOKEN | String | 是   | OpenAPI认证token      |
| Authorization         | String | 是   | 制品库basic认证请求头 |

#### 路径参数

无

#### 查询参数

无

#### 请求体

无

#### 请求示例

```http
DELETE {{devops-domain}}/api/open/CPack/repo-repository/api/log/event/type
```




### 返回结果

| 字段    | 说明           |
| ------- | -------------- |
| code    | 返回码         |
| message | 错误信息       |
| data    | 事件类型及说明 |
| traceId | 链路追踪id     |
