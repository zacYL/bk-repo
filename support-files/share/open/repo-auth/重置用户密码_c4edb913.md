# 重置用户密码
功能描述：重置用户密码

### 请求地址
```
/api/user/reset/{uid}
```

### 请求方法
`POST`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| uid     | string   | true       | 需要重置密码用户id |

#### 查询参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| newPwd     | string   | true       | 重置密码, 为空重设为默认密码 |


#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
