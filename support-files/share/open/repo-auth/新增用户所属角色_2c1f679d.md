# 新增用户所属角色
功能描述：新增用户所属角色

### 请求地址
```
/api/user/role/{uid}/{rid}
```

### 请求方法
`POST`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| uid     | string   | true       | 用户id |
| rid     | string   | true       | 用户角色id |



#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
