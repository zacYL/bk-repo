# 编辑项目
功能描述：编辑项目

### 请求地址
```
/api/project/{name}
```

### 请求方法
`PUT`
### 请求参数
#### 路径参数

| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| name     | string   | true       | 项目ID |



#### 请求体参数
| 字段 | 类型 | 必填 | 描述 |
| -------- | -------- | -------- | -------- |
| description     | string   | 描述 |
| displayName     | string   | 显示名 |
