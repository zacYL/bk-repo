# 查询许可信息

功能描述：查询许可信息

### 请求地址

```
/repo-repository/api/license
```

### 请求方法

`GET`

### 返回结果

| 字段      | 说明     |
|---------|--------|
| code    | 返回码    |
| message | 错误信息   |
| data    | 数据     |
| traceId | 链路追踪id |

#### 响应体示例

```json
{
  "code": 0,
  "message": null,
  "data": {
    "productKey": "CPack",
    "productName": "制品管理平台产品",
    "versionType": "ENTERPRISE",
    "modules": "ANALYSE,DELIVERY,SEARCH,LIBRARY",
    "applyUserNumber": 2147483647,
    "applyNodeNumber": 1,
    "validityTime": "2024-05-12 00:00:00",
    "subscribeType": "SUBSCRIBE"
  },
  "traceId": ""
}
```

#### data 字段说明

许可信息

| 字段              | 说明         |
|-----------------|------------|
| productKey      | 产品key      |
| productName     | 产品名称       |
| versionType     | 产品版本       |
| modules         | 产品包含的模块    |
| applyUserNumber | 使用证书的最大用户数 |
| applyNodeNumber | 使用证书的最大节点数 |
| validityTime    | 证书失效日期     |
| subscribeType   | 订阅类型       |
