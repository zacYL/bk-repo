# 更新仓库

功能描述：更新仓库

### 请求地址

```
/repo-repository/api/repo/update/{projectId}/{repoName}
```

### 请求方法

`POST`

### 请求参数

#### 路径参数

| 字段        | 类型     | 必填  | 描述   |
|-----------|--------|-----|------|
| projectId | String | 是   | 所属项目 |
| repoName  | String | 是   | 仓库名称 |

#### 请求体参数

| 字段            | 类型      | 必填  | 描述   |
|---------------|---------|-----|------|
| public        | Boolean | 否   | 是否公开 |
| description   | String  | 否   | 简要描述 |
| configuration | Object  | 否   | 扩展信息 |
| quota         | Long    | 否   | 仓库配额 |
| coverStrategy | String  | 否   | 覆盖策略 |

### 返回结果

| 字段      | 说明        |
|---------|-----------|
| code    | 返回码       |
| message | 错误信息      |
| data    | 数据, 为null |
| traceId | 链路追踪id    |

#### 响应体示例

```json
{
  "code": 0,
  "message": null,
  "data": null,
  "traceId": null
}
```
